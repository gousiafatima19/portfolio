# 🌸 Gousia Fatima Portfolio

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.  
This portfolio showcases my skills, projects, education, achievements, and contact information in a clean, elegant, and minimal design.

# ✨ Features

- Responsive modern UI
- Smooth scrolling navigation
- Elegant typography and aesthetic layout
- Animated sections using Intersection Observer
- Fully responsive design for desktop and mobile
- Project showcase section
- Skills & achievements display
- Education timeline
- Resume download button
- Contact links (GitHub, LinkedIn, Email)
- Clean and minimal portfolio design
